# Wortal Web Section

A pixel-faithful implementation of the Wortal mobile app landing section built from Figma design.

Live Demo
https://timely-vacherin-c77c0c.netlify.app/

Built With
- HTML5, CSS3, Vanilla JavaScript
- Google Fonts (Outfit)
- No frameworks or dependencies

Features
- Responsive design (mobile, tablet, desktop)
- Interactive feature tabs
- Animated phone mockups
- Pixel-matched to Figma design
